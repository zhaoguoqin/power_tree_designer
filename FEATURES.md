# 电源树设计器 — 功能说明

## 概述

**电源树设计器 (Power Tree Designer)** 是一款桌面应用，用于嵌入式系统电源树的可视化设计、功率计算与分析。用户通过拖拽方式构建从输入电源到各负载的供电路径，软件自动逐级计算电压、电流、功率和效率。

基于 Python + PySide6 开发，可打包为独立 exe，无需安装 Python 环境即可运行。

## 核心功能

### 1. 电源树拓扑图编辑器

- **拖拽式建树**：从模块库拖拽电源模块到画布，自动建立父子节点关系
- **节点类型**：支持输入电源、Buck 降压、Boost 升压、Buck-Boost 升降压、LDO 线性稳压、负载，以及自定义类型
- **90° 直角连线**：父子节点间使用正交路径连接，清晰直观
- **点击连线断开**：悬停连线变红，点击弹出确认对话框断开父子关系
- **手动重连**：未连接节点可通过属性面板"连接到父节点..."按钮重新挂载
- **画布操作**：滚轮缩放、中键拖拽平移、Delete 键删除节点、Ctrl+L 自动布局

### 2. 电源模块管理系统

- **预置模块库**：内置常用 Buck/Boost/LDO/负载模板
- **自定义模块**：支持创建任意类型的电源模块，参数持久化为 JSON 文件
- **模块字段**：
  - 名称、类型（预设或自定义）、芯片型号
  - 输入电压范围、输出电压（可设可调范围）
  - 最大输出电流、典型效率、静态电流
  - 效率曲线（多负载点插值）
  - 数据手册来源 URL
- **模块管理**：新建、编辑、删除、复制、导入/导出

### 3. 实时功率计算引擎

- **计算方式**：后向传播（负载 → 电源），逐级累加
- **计算内容**：
  - 各节点：Vin、Vout、Iin、Iout、Pin、Pout、Ploss、η
  - 系统级：总输入功率、总输出功率、总损耗、系统效率
- **效率计算**（优先级从高到低）：
  1. 用户手动覆盖的效率值
  2. 模块效率曲线（按负载比例插值）
  3. 模块固定效率值
  4. 默认值 90%
- **LDO 特殊处理**：未覆盖效率时使用物理公式 `Iin = Iout + Iq`，`Ploss = Vin × Iin - Vout × Iout`
- **参数实时更新**：修改任意节点参数后，点击"应用"立即重新计算并刷新显示

### 4. 节点可视化

- **颜色编码**：蓝色(输入)、绿色(Buck)、橙色(Boost)、紫色(Buck-Boost)、青色(LDO)、红色(负载)
- **信息显示**：每个节点卡片显示类型、芯片型号、名称、电压、功率、效率
- **自适应宽度**：根据芯片名、类型名长度自动调整节点宽度
- **选中高亮**：选中节点蓝色边框加粗

### 5. 项目文件管理

- **文件格式**：`.pwt`（JSON 格式，包含完整树结构和节点参数）
- **操作**：新建、打开、保存、另存为
- **最近文件**：自动记录最近打开的 10 个项目文件

### 6. 导出功能

- **CSV 报表**：导出所有节点的完整计算数据（电压、电流、功率、效率、损耗）
- **PNG 截图**：导出画布当前视图为图片
- **SVG 矢量图**：导出画布为矢量图形

## 技术架构

| 层级 | 技术 |
|------|------|
| 语言 | Python 3.9+ |
| GUI 框架 | PySide6 (Qt 6) |
| 可视化 | QGraphicsView / QGraphicsScene |
| 数据格式 | JSON（模块定义 + 项目文件） |
| 打包 | PyInstaller |

## 项目结构

```
power-tree-designer/
├── src/
│   ├── main.py                    # 入口
│   ├── models/
│   │   ├── power_module.py        # 电源模块数据模型
│   │   └── tree_node.py           # 树节点数据模型
│   ├── core/
│   │   ├── power_tree.py          # 电源树数据结构
│   │   ├── calculator.py          # 功率计算引擎
│   │   ├── module_manager.py      # 模块管理器
│   │   └── project_manager.py     # 项目管理器
│   └── ui/
│       ├── main_window.py         # 主窗口
│       ├── tree_canvas.py         # 画布组件（含自动布局算法）
│       ├── tree_node_item.py      # 节点图形项
│       ├── tree_edge_item.py      # 连线图形项
│       ├── module_library.py      # 模块库面板
│       ├── property_panel.py      # 属性编辑面板
│       └── dialogs/
│           ├── module_editor.py   # 模块编辑器对话框
│           └── export_dialog.py   # 导出对话框
├── modules/                       # 预置模块库（JSON 文件）
├── tests/                         # 单元测试
├── requirements.txt
├── build.py                       # PyInstaller 打包脚本
└── FEATURES.md                    # 本文件
```
