# 电源树设计器 — 用户操作指南

## 快速开始

### 安装与运行

**方式一：运行 exe（推荐）**

直接双击 `电源树设计器.exe`，无需安装任何依赖。

**方式二：从源码运行**

```bash
pip install -r requirements.txt
python -m src.main
```

## 基本操作

### 构建电源树

1. **添加根节点**：从左侧模块库拖拽"输入电源"（如 DC_12V_Input）到画布
2. **添加转换器**：拖拽 Buck/Boost/LDO 模块到画布，自动连接到根节点
3. **添加负载**：拖拽 Load 模块，自动连接
4. **指定父节点**：先在画布上**点击选中**目标父节点，再拖入模块

### 编辑节点属性

1. **点击画布上的节点**选中它
2. 右侧属性面板显示该节点参数：
   - **名称**：节点的显示名称
   - **类型**：模块类型（只读）
   - **芯片**：芯片型号，可直接修改（如 LMS1117）
   - **输入电压**：根节点可编辑；非根节点由父节点决定（只读）
   - **输出电压**：可修改
   - **输出电流**：负载节点可编辑；转换器由子节点决定（只读）
   - **手动覆盖效率**：勾选后可自定义效率百分比
3. 修改完成后点击 **"应用"**，电源树自动重新计算

### 管理连接

| 操作 | 方法 |
|------|------|
| 断开连接 | 悬停连线 → 变红 → 点击 → 确认 |
| 重新连接 | 选中未连接节点 → 属性面板点击"连接到父节点..." → 选择目标 |
| 删除节点 | 选中节点 → 按 Delete 键 / 属性面板点击"删除节点" |

### 模块管理

**新建模块**：
1. 点击模块库下方"新建"按钮（或右键 → "新建模块"）
2. 填写：
   - 模块名称、类型（可选自定义类型名）
   - 芯片型号（如 TPS5430）
   - 输入/输出电压范围、最大电流、效率
   - 数据手册 URL（可选）
3. 点击 OK 保存，模块自动加入库中

**编辑/删除/复制模块**：右键点击模块库中的模块项

### 画布操作

| 操作 | 方法 |
|------|------|
| 新建 | 文件 → 新建项目 (`Ctrl+N`) |
| 打开 | 文件 → 打开项目 (`Ctrl+O`) |
| 保存 | 文件 → 保存 (`Ctrl+S`) |
| 另存为 | 文件 → 另存为 (`Ctrl+Shift+S`) |
| 最近文件 | 文件 → 最近打开 |

### 导出

1. 文件 → 导出 (`Ctrl+E`)
2. 选择导出目录和格式：
   - **CSV**：计算数据报表
   - **PNG**：画布截图
   - **SVG**：矢量图
3. 点击 OK

## 典型用例

### 用例 1：简单 MCU 系统

> 12V 适配器 → 5V Buck → 3.3V LDO → MCU + 传感器

**步骤**：
1. 拖入 `DC_12V_Input`
2. 点击选中 12V 节点，拖入 `BUCK_12V_to_5V_3A`
3. 点击选中 Buck 节点，拖入 `LDO_5V_to_3V3_1A`
4. 点击选中 LDO 节点，拖入 `Load_MCU` 和 `Load_Sensor`
5. 选中 MCU 负载，在属性面板设置负载电流为 0.1A
6. 选中传感器负载，设置电流为 0.05A
7. 按 `Ctrl+L` 自动布局
8. 查看状态栏：总输入功率、系统效率

### 用例 2：多电压轨 FPGA 系统

> 12V → 5V Buck → 3.3V Buck → FPGA Core
> 12V → 3.3V Buck → FPGA IO
> 12V → 1.8V Buck → DDR

**步骤**：
1. 拖入 `DC_12V_Input`
2. 拖入多个 Buck 模块（它们会自动连到 12V）
3. 对每个 Buck 设置不同的输出电压
4. 添加各类负载
5. 如需改连接：选中子节点 → "连接到父节点..."

## 效率计算说明

软件使用**后向传播**算法：

1. 从负载开始：`P_out = V_out × I_out`
2. 向上一级：`P_in = P_out / η`，`P_loss = P_in - P_out`
3. 父节点的 `P_out = Σ(子节点 P_in)`
4. 重复直到根节点

效率 η 的取值优先级：
1. 属性面板手动覆盖（如果勾选）
2. 模块效率曲线插值（如果定义了曲线）
3. 模块固定效率值
4. 默认 90%

**LDO 特殊公式**（未手动覆盖效率时）：
- `I_in = I_out + I_q`
- `P_in = V_in × I_in`
- 效率 = `P_out / P_in`
